# Electron

The Electron-based shell for DFN v2.

Follow instructions in the root directory for setup.

## Start it

```bash
pnpm start
```

## Start it with live reload

```bash
pnpm run start:dev
```

## Compile TypeScript code

```bash
pnpm run compile
```

## Build the static web page

```bash
pnpm run build
```

Static web page is output to `./dist`.
